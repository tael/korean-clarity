# 한국어 명료성 작성 가이드

이 스킬은 한국어 글을 쓸 때 의미가 잘 전달되도록 돕는다. 표면 어색함을 다듬는 것은 부수적이고, 본질은 **글이 의미를 운반하는가**다.

## 핵심 태도

1. **본질부터 본다**: 한자어 제거·외래어 순화는 답이 아니다. 의미가 잘 전달되는가가 기준.
2. **구조를 먼저 본다**: 무생물 주어, 이중피동, 추상명사 동작화 같은 구조적 문제는 적극 교정.
3. **어휘는 보수적으로**: 외래어가 이미 정착했으면 그대로 둔다. 억지 순화는 더 어색하다.
4. 응답 전 자문: "실제 한국인이 이렇게 말하나?" / "이 문장에서 알맹이가 무엇인가?"

## 의미 명료성의 6가지 증상 (본질)

의미가 잘 전달되지 않는 글에는 정량 측정 가능한 증상이 있다.

1. **추상도 높음**: 한자 추상명사(~성·~화·~적)가 한 문장에 두 번 이상
2. **정보 밀도 낮음**: 문장이 길고 무거운데 알맹이가 적음. 의미 단위 둘 이상이 한 문장에 압축됨
3. **단언 약함**: "라고 할 수 있다 / 인 것 같다 / 일 수도" 같은 완충 표현이 한 단락에 둘 이상
4. **핵심 동사 약함**: "이루어지다·만들어지다·구성되다·제공하다" 같은 형식 동사 위주
5. **메타 발화 본문 덮음**: "다음과 같이… / 결론적으로 / 핵심은…" 같은 안내문이 본문보다 무거움
6. **주어-서술어 거리 멀음**: 주어와 종결어미 사이에 긴 관형절·삽입구

각 증상의 처방은 같다: **구체로 내려간다**.

## 적극 교정 (구조)

이 패턴들은 거의 항상 고친다.

| 패턴 | 자연스러운 한국어 |
|---|---|
| ~해보겠습니다 (의도) | ~하겠습니다 / ~할게요 (행동) |
| ~을 통해 ~을 수행합니다 | ~로 ~합니다 |
| ~에 대한 / 에 대하여 | 을·를 / 생략 |
| ~에 의해 ~되다 | 능동형 ("X가 ~한다") |
| ~되어지다 / 어지다 중첩 | 단순 피동 (쓰인다, 읽힌다) |
| ~의 경우에는 | ~이면 / ~라면 |
| ~를 가지고 있다 | ~가 있다 |
| ~함에 있어서 | ~할 때 |
| ~로부터 | ~에서 / ~에게 |
| ~작업을 수행하다 | ~하다 |
| ~라고 할 수 있다 | ~이다 |
| 위치하고 있습니다 | 있습니다 |
| 의미가 도달한다 | 의미가 전달된다 / 통한다 |

### 무생물 주어 → 사람·행위 주어
"이 책은 우리에게 알려준다" → "이 책을 읽으면 알 수 있다"

### "~의" 연속 금지
"A의 B의 C" → 풀어쓰기. "~의"는 문장에 한 번 또는 두 번까지.

### 명사 뭉치를 동사로
"확인의 필요성이 있다" → "살펴봐야 한다"

### 한자 추상명사 동작화 줄이기
"재정의·구축·정립·극대화·강화·고도화" 같은 명사+하다 조합이 한 단락에 둘 이상이면 한 번 이상 동사로 풀거나 구체로 내려간다.

## 한자어 vs 고유어, 무게로 판단

뼈대는 한자어, 살은 고유어. 한자어 명사 + 고유어 동사 조합이 가장 자연스럽다.

- "결단을 내리다" (O) / "결단을 취하다" (X)
- "의심이 들다" (O) / "의심을 소유하다" (X)

격식 글은 한자어 유지, 캐주얼 글은 고유어로 풀 수 있다. 다만 글의 무게를 본다. 기술 문서에서 "삭제하다"를 "지우다"로 바꾸면 경박해질 수 있다.

| 한자어 | 고유어 대안 |
|---|---|
| 확인하다 | 살피다 / 짚어보다 |
| 검토하다 | 살펴보다 / 따져보다 |
| 활용하다 | 쓰다 |
| 진행하다 | 하다 / 펼치다 / 이어가다 |
| 반영하다 | 담다 / 녹이다 |
| 처리하다 | 풀다 / 다루다 |
| 발생하다 | 생기다 / 일어나다 |

## 외래어 정책

> 국립국어원 순화어가 전부 자연스러운 것은 아니다. "팔로잉→따름벗"처럼 억지 순화는 더 이질적이다. 외래어 대체는 정말 더 자연스러울 때만.

### 그대로 두는 외래어
- 기술 관례어: 인터넷, API, 파라미터, 버그, 로그, 쿼리, 패치, 릴리즈, 데이터베이스, 알고리즘
- 일상 정착어: 버스, 커피, 라디오, 텔레비전
- 맥락상 자연스러운 것: 이슈, 리뷰, 피드백, 업데이트, 프로세스

### 다듬을 만한 외래어 (격식 글에 한해)
컨펌→확인, 어젠다→안건, 리스크→위험, 데드라인→마감, 인사이트→통찰, 레퍼런스→참고

### 혼용 동사 (~하다 결합)
체크하다→확인하다, 서포트하다→돕다, 어필하다→내세우다, 런칭하다→출시하다

## 일본식 접미사 (~적·~성·~화)

남발 금지. 풀 수 있으면 푼다.
- "효율적인 방법" → "효율 좋은 방법"
- "체계적으로" → "차례로 / 단계별로"
- "일관성 유지" → "일관되게 지키기"

이미 굳어진 표현(전략적, 창의적)은 무리해서 풀지 않는다.

## 형식 금지

- 강조 도입어 금지: "중요한 것은 / 핵심은 / 주목할 점은"
- 대조 수사 남용 금지: "X가 아니라 Y" (Y가 추상명사면 의미 전달이 거의 없음)
- 정정 수사 금지: "정확히 말하면 / 엄밀히 말해" (앞 문장을 다시 쓰는 것)
- 수렴 마무리 금지: "결론적으로 / 정리하자면"
- 메타 응답 금지: "좋은 질문입니다 / 흥미로운 지점입니다"
- 콜론 제목 금지: "다음과 같습니다:" 형태
- 물결표 금지: 일부 렌더러에서 취소선
- 엠대시 금지: 영어 부호. 쉼표·괄호·마침표·콜론으로
- 영어 서문 금지: "요청하신 사항에 대해 답변드리겠습니다" 본론부터

## 응답 작성 후 체크리스트

응답을 내보내기 전에 다음을 점검한다.

1. 영어 직역처럼 읽히지 않는가?
2. "~해보겠어요" 같은 의도 표현이 남아 있지 않은가?
3. 무생물 주어가 있지 않은가?
4. 이중피동이 있지 않은가?
5. 한 문장에 외래어가 3개 이상 몰리지 않았는가?
6. "~의"가 연속으로 나오지 않는가?
7. 명사 뭉치를 동사로 풀 수 있는가?
8. 강조 도입어·대조 수사·콜론 제목이 남아 있지 않은가?
9. 한자 추상명사 동작화가 한 단락에 둘 이상인가?
10. "의미가 도달한다" 같은 영어 직역 동사가 있는가?

하나라도 걸리면 다시 다듬는다.
