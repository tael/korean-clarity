# AI 클리셰 가드 (검토용)

이 스킬은 작성된 한국어 글에서 AI 클리셰·번역체·격 어색함·의미 명료성 위반을 찾아낸다. 작성 기준은 `korean-clarity` 스킬과 동일하다. 같은 룰을 "생성"이 아닌 "검토" 관점에서 적용한다.

## 검토 태도

1. **AI 특유 클리셰는 강하게 지적** — 강조 도입어, 대조 수사, 정정 수사, 메타 발화, 콜론 제목
2. **구조 번역체는 강하게 지적** — 이중피동, 무생물 주어, "~에 의해" 수동
3. **어휘 교체는 제안일 뿐** — 원어 차음(외래어 그대로)이 자연스러울 수 있음을 인정
4. **의미 명료성을 정량으로 본다** — 표면 위반이 적어도 추상도가 높으면 명료성 낮음을 지적

## 검토 카테고리

| 카테고리 | 무엇을 보는가 |
|---|---|
| A 수사 클리셰 | "핵심은", "X가 아니라 Y", "결론적으로", "정확히 말하면", 메타 응답 |
| B 포맷 클리셰 | "다음과 같습니다:", 글머리 남용, 강조 마크다운 남발, 이모지, 엠대시·물결표 |
| C 구조 번역체 | 무생물 주어, 이중피동, "~에 의해" 수동, 긴 관형절 앞쏠림 |
| D 어휘 직역 | "위치하고 있다", "~에 대한", "~함에 있어서", "합성기/처리기" 류 단어 직역 |
| E 격 어색함 | "~의" 연속, ~적·~성·~화 남발, 명사 뭉치, 의도형 어미, "~들" 남발 |
| F 의미 명료성 | 추상도, 정보 밀도, 단언 강도, 핵심 동사 비율, 메타 비율, 주어-서술어 거리 |

## 심각도

```
[높음]  분명한 번역체·클리셰. 반드시 고친다.
        - 이중피동, "~에 의해" 수동, "가지고 있다", "함에 있어서"
        - 강조 도입어, 수렴 마무리

[중간]  맥락에 따라 허용 가능하지만 다듬는 게 좋음.
        - 콜론 제목, "~에 대한", "~의" 연속
        - 추상명사 동작화 (재정의·구축·정립)
        - 정정 수사

[낮음]  스타일 제안.
        - 한자어를 고유어로 풀 수 있는 자리
        - 한 문장에 외래어 3개 이상
        - ~적·~성·~화 (이미 굳은 것은 제외)
```

```

### 심각도 높음 (반드시 교정)

2. ...

### 심각도 중간 (맥락에 따라 선택)

2. ...

### 심각도 낮음 (스타일 제안)
```

## 외래어 검토 시 유의

외래어를 발견했다고 무조건 순화하라고 제안하지 않는다.

- "이슈", "리뷰", "피드백", "업데이트", "프로세스"는 이미 한국어 리듬에 녹아 있다
- "API", "파라미터", "버그", "로그" 같은 기술 관례어는 그대로가 정답
- "컨펌→확인"처럼 대체가 명확히 더 자연스러울 때만 제안한다

## 의미 명료성 점검 (가장 자주 놓치는 영역)

표면 위반이 적어도 명료성이 낮을 수 있다. 다음을 확인.

- **한 문장에 한자 추상명사가 셋 이상인가?** "효율성의 극대화를 위한 프로세스의 정립" 같은 줄.
- **메타 발화 비율이 본문보다 많은가?** "다음과 같이 설명드리겠습니다", "한 가지 분명한 것은", "정리하자면"
- **완충 표현이 한 단락에 두 번 이상 나오는가?** "라고 할 수 있다 / 인 것 같다"
- **핵심 동사가 모두 형식 동사인가?** "이루어진다·만들어진다·제공된다·구성된다"

이 중 둘 이상에 해당하면 명료성 점수가 낮다고 보고 처방한다.
